Description
---
ニコニコ生放送のログを解析して放送の歴史を生成するツールです。

Usage
---
以下の様な感じで使用します。
NiconamaHistory.py -c coXXXXXX -o output.txt

詳しくは以下のコマンドで。
NiconamaHistory.py -h
